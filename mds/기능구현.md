CMS 생성형 게시판
================
소개
----
cms-ino는 사용자가 쉽게 CMS(콘텐츠 관리 시스템) 기반의 **페이지와 게시판**을
분리하여 생성하고 관리할 수 있도록 도와주는 도구입니다. 이 도구는 다양한 기능과 사용자 친화적인 인터페이스를 제공하여, 비전문가도 손쉽게 웹사이트와 게시판을 구축할 수 있습니다.

### 페이지와 게시판의 차이점
- **페이지(Page)**: 정적 콘텐츠를 관리하는 단위입니다. 회사 소개, 서비스 안내, 이용약관 등 자주 변경되지 않는 콘텐츠를 관리합니다.
  - 기본 템플릿 제공: 메인 페이지, 소개 페이지, 서브 페이지 등 다양한 기본 템플릿 제공
  - 커스텀 퍼블리싱: HTML/CSS/JS를 직접 작성하여 원하는 디자인 적용 가능
  
- **게시판(Board)**: 동적 콘텐츠를 관리하는 단위입니다. 공지사항, 자유게시판, Q&A 등 사용자들이 지속적으로 글을 작성하고 소통하는 공간입니다.
  - 기본 템플릿 제공: 리스트형, 갤러리형, 카드형 등 다양한 게시판 레이아웃 제공
  - 커스텀 퍼블리싱: 게시판 스킨(Skin)을 직접 제작하여 독창적인 디자인 구현 가능

관리자 주요 기능
--------
> 관리자 페이지는 GNB(Global Navigation Bar), LNB(Local Navigation Bar), SNB(Side Navigation Bar) 구조로 구성됩니다.

### 1. 메뉴 관리 (사용자)
사용자에게 노출되는 사이트 메뉴를 구성하고 관리합니다.
- **메뉴 구조 관리**: 계층형 메뉴 생성 및 드래그 앤 드롭으로 순서 변경
- **메뉴 타입 설정**: 페이지 링크, 게시판 링크, 외부 링크, 메뉴 그룹
- **메뉴 권한 설정**: 회원등급별 메뉴 노출 제어
- **다국어 메뉴**: 언어별 메뉴명 설정
- **메뉴 아이콘**: 메뉴에 아이콘 이미지 또는 폰트 아이콘 적용

### 2. 메인 관리
사이트 메인 페이지를 구성하고 관리합니다.
- **메인 레이아웃 설정**: 기본 제공 템플릿 선택 또는 커스텀 퍼블리싱 적용
- **섹션 관리**: 메인 페이지의 각 섹션(슬라이더, 공지사항, 제품 등) 구성
- **비주얼 에디터**: 드래그 앤 드롭 방식의 메인 페이지 빌더
- **미리보기**: 실시간 메인 페이지 미리보기 및 반응형 테스트

### 3. 콘텐츠 관리
사이트의 핵심 콘텐츠를 관리합니다.

#### 3-1. 페이지 관리
- **페이지 생성/수정/삭제**: 정적 콘텐츠 페이지 관리
- **기본 템플릿 제공**: 메인, 서브, 랜딩 페이지 등 다양한 템플릿
- **커스텀 퍼블리싱**: HTML/CSS/JS 직접 편집 및 파일 업로드
- **페이지 빌더**: 드래그 앤 드롭 방식의 위지윅 에디터
- **페이지 계층 구조**: 메뉴 구조에 맞춘 페이지 계층 관리
- **SEO 설정**: 메타태그, 키워드, 설명 등 SEO 최적화
- **버전 관리**: 페이지 수정 이력 관리 및 롤백 기능

#### 3-2. 게시판 관리
- **게시판 생성/수정/삭제**: 다중 게시판 생성 및 관리
- **게시판 타입**: 일반, 갤러리, Q&A, FAQ, 자료실 등
- **기본 템플릿 제공**: 리스트형, 갤러리형, 카드형, 타임라인형, FAQ형
- **커스텀 스킨 시스템**: 목록/보기/쓰기/수정 페이지별 스킨 커스터마이징
- **게시판 설정**: 
  - 권한 설정 (읽기, 쓰기, 댓글, 다운로드)
  - 페이징 설정 (목록 수, 페이지 수)
  - 분류(카테고리) 설정
  - 첨부파일 설정 (허용 확장자, 용량 제한)
  - 에디터 설정 (위지윅, 마크다운, 텍스트)
- **동적 필드 적용**: 게시판별 커스텀 필드 설정 (시스템 관리 연동)

#### 3-3. 게시글 관리
- **게시글 목록 조회**: 전체 게시판의 게시글 통합 관리
- **게시글 검색/필터링**: 게시판별, 작성자별, 기간별 검색
- **게시글 수정/삭제**: 관리자 권한으로 게시글 직접 수정 및 삭제
- **게시글 이동/복사**: 게시판 간 게시글 이동 및 복사
- **공지사항 설정**: 상단 고정, 전체 공지 설정
- **게시글 상태 관리**: 공개, 비공개, 임시저장, 예약 발행
- **댓글 관리**: 댓글 목록 조회, 수정, 삭제, 블라인드 처리

### 4. 회원 관리
사이트 회원 및 관리자를 관리합니다.

#### 4-1. 사용자 관리
- **회원 목록 조회**: 전체 회원 목록 및 상세 정보 조회
- **회원 검색**: 아이디, 이름, 이메일, 가입일 등으로 검색
- **회원 등급 관리**: 회원 등급 설정 및 변경
- **회원 상태 관리**: 정상, 휴면, 정지, 탈퇴 상태 관리
- **회원 정보 수정**: 관리자 권한으로 회원 정보 수정
- **대량 처리**: 엑셀 다운로드, 일괄 등급 변경, 이메일 발송

#### 4-2. 관리자 관리
- **관리자 계정 관리**: 관리자 계정 생성, 수정, 삭제
- **관리자 권한 그룹 연동**: 권한 그룹별 관리자 배정
- **접속 이력 관리**: 관리자 로그인 이력 및 활동 로그

#### 4-3. 관리자 권한 관리
- **권한 그룹 생성**: 역할별 권한 그룹 생성 (최고관리자, 운영자, 편집자 등)
- **메뉴별 권한 설정**: GNB/LNB/SNB 메뉴별 접근 권한 설정
- **기능별 권한 설정**: 읽기, 쓰기, 수정, 삭제 권한 세분화
- **IP 접근 제한**: 특정 IP에서만 관리자 페이지 접근 허용

### 5. 사이트 관리
사이트 운영에 필요한 부가 기능을 관리합니다.

#### 5-1. 배너 관리
- **배너 등록/수정/삭제**: 이미지 또는 HTML 배너 등록
- **배너 그룹 관리**: 위치별 배너 그룹 설정 (상단, 사이드, 하단 등)
- **노출 설정**: 게시 기간, 노출 페이지, 타겟팅 설정
- **배너 통계**: 노출 수, 클릭 수, 클릭률(CTR) 확인

#### 5-2. 팝업 관리
- **팝업 등록/수정/삭제**: 레이어 팝업, 모달 팝업 등록
- **팝업 디자인**: HTML/CSS 에디터로 팝업 디자인 커스터마이징
- **노출 설정**: 게시 기간, 노출 페이지, 노출 빈도 설정
- **팝업 옵션**: 오늘 하루 보지 않기, 위치 고정, 드래그 가능 설정

#### 5-3. 다국어 관리
- **언어 설정**: 지원 언어 추가 및 기본 언어 설정
- **다국어 리소스 관리**: 시스템 메시지 및 레이블 다국어 등록
- **콘텐츠 다국어 관리**: 페이지, 게시판, 메뉴 등 콘텐츠별 다국어 버전 관리
- **자동 번역 연동**: 외부 번역 API 연동 (선택 사항)

#### 5-4. 파일 관리
- **파일 탐색기**: 업로드된 파일 및 폴더 구조 관리
- **파일 업로드**: 단일/다중 파일 업로드 및 드래그 앤 드롭
- **파일 검색**: 파일명, 확장자, 업로드일로 검색
- **스토리지 관리**: 저장 용량 확인 및 정리
- **파일 권한**: 파일별 접근 권한 설정

### 6. 통계
사이트 운영 현황을 분석합니다.

#### 6-1. 사용자 통계
- **방문자 통계**: 일별, 주별, 월별 방문자 수 및 페이지뷰
- **유입 경로 분석**: 검색엔진, 직접 방문, 외부 링크 등
- **디바이스 분석**: PC, 모바일, 태블릿 비율
- **회원 가입 통계**: 기간별 신규 회원 가입 추이
- **회원 활동 통계**: 로그인 빈도, 활동 회원 비율

#### 6-2. 게시글 통계
- **게시글 현황**: 게시판별 게시글 수 및 증가 추이
- **인기 게시글**: 조회수, 댓글수, 추천수 기준 인기 게시글
- **게시글 작성 통계**: 기간별 게시글 작성 추이
- **댓글 통계**: 게시판별 댓글 수 및 활동 현황

### 7. 시스템 관리
시스템 설정 및 고급 기능을 관리합니다.

#### 7-1. 게시판 템플릿 관리
- **템플릿 등록/수정/삭제**: 재사용 가능한 게시판 템플릿 생성
- **기본 템플릿 커스터마이징**: 제공되는 기본 템플릿 수정
- **템플릿 스킨 관리**: 목록/보기/쓰기/수정 페이지 스킨 파일 관리
- **템플릿 미리보기**: 템플릿 적용 결과 미리보기
- **템플릿 내보내기/가져오기**: 템플릿 파일 백업 및 복원

#### 7-2. 게시판 동적 필드 관리
- **필드 생성**: 게시판에 추가할 커스텀 필드 생성
- **필드 타입**: 텍스트, 숫자, 날짜, 선택(셀렉트박스), 체크박스, 라디오, 파일 등
- **필드 검증**: 필수 입력, 최소/최대 길이, 정규식 검증 설정
- **필드 레이아웃**: 필드 배치 순서 및 표시 옵션 설정
- **필드 권한**: 필드별 읽기/쓰기 권한 설정


관리자 사용법
--------
### 관리자 페이지 접속
1. 웹 브라우저에서 관리자 페이지 URL로 접속합니다. (예: https://yourdomain.com/admin)
2. 관리자 계정으로 로그인합니다.
3. 대시보드에서 전체 사이트 현황을 확인합니다.

### GNB/LNB/SNB 구조
- **GNB (Global Navigation Bar)**: 최상위 메뉴 (메뉴관리, 메인관리, 콘텐츠관리 등)
- **LNB (Local Navigation Bar)**: 하위 메뉴 (페이지 관리, 게시판 관리, 게시글 관리 등)
- **SNB (Side Navigation Bar)**: 상세 기능 및 필터 옵션

### 1. 메뉴 관리 (사용자)
```
GNB: 메뉴 관리
```
1. 좌측 GNB에서 "메뉴 관리" 클릭
2. "새 메뉴 추가" 버튼으로 메뉴 생성
3. 메뉴 타입 선택 (페이지 링크, 게시판 링크, 외부 링크, 메뉴 그룹)
4. 드래그 앤 드롭으로 메뉴 순서 및 계층 조정

### 2. 메인 관리
```
GNB: 메인 관리
```
1. "메인 관리" 메뉴 클릭
2. 레이아웃 템플릿 선택 또는 커스텀 퍼블리싱
3. 섹션별 콘텐츠 편집 (슬라이더, 공지사항, 제품 등)
4. 드래그 앤 드롭으로 섹션 순서 조정
5. 미리보기로 확인 후 저장

### 3. 콘텐츠 관리
```
GNB: 콘텐츠 관리
LNB: 페이지 관리 / 게시판 관리 / 게시글 관리
```

#### 페이지 관리
1. "콘텐츠 관리 > 페이지 관리" 클릭
2. "새 페이지 생성" 버튼 클릭
3. 템플릿 선택 (기본 템플릿 또는 커스텀 퍼블리싱)
4. 페이지 빌더로 콘텐츠 작성 및 배치
5. SEO 설정 (메타태그, 키워드 등)
6. 미리보기 확인 후 발행

#### 게시판 관리
1. "콘텐츠 관리 > 게시판 관리" 클릭
2. "새 게시판 생성" 버튼 클릭
3. 게시판 기본 정보 입력 (이름, 설명, 타입)
4. 템플릿 선택 (리스트형, 갤러리형 등) 또는 커스텀 스킨 적용
5. 게시판 설정 구성:
   - 권한 설정 (읽기, 쓰기, 댓글 등)
   - 페이징 설정
   - 카테고리 설정
   - 첨부파일 설정
6. 동적 필드 추가 (필요시)
7. 저장 후 메뉴와 연결

#### 게시글 관리
1. "콘텐츠 관리 > 게시글 관리" 클릭
2. 검색 필터로 원하는 게시글 조회 (게시판, 작성자, 기간 등)
3. 게시글 수정/삭제/이동/복사
4. 공지사항 설정 또는 상태 변경 (공개, 비공개 등)
5. 댓글 관리 (수정, 삭제, 블라인드)

### 4. 회원 관리
```
GNB: 회원 관리
LNB: 사용자 / 관리자 / 관리자 권한
```

#### 사용자 관리
1. "회원 관리 > 사용자" 클릭
2. 회원 목록에서 검색 및 필터링
3. 회원 상세 정보 조회 및 수정
4. 회원 등급 변경 또는 상태 관리 (정지, 휴면 등)
5. 대량 처리 (엑셀 다운로드, 이메일 발송 등)

#### 관리자 관리
1. "회원 관리 > 관리자" 클릭
2. "새 관리자 추가" 버튼 클릭
3. 관리자 정보 입력 및 권한 그룹 선택
4. 저장 후 관리자 계정 활성화

#### 관리자 권한 관리
1. "회원 관리 > 관리자 권한" 클릭
2. "새 권한 그룹 생성" 버튼 클릭
3. 그룹명 입력 (예: 운영자, 편집자 등)
4. 메뉴별 접근 권한 체크 (GNB/LNB 단위)
5. 기능별 권한 설정 (읽기, 쓰기, 수정, 삭제)
6. 저장 후 관리자에게 권한 그룹 할당

### 5. 사이트 관리
```
GNB: 사이트 관리
LNB: 배너 / 팝업 / 다국어 / 파일 관리
```

#### 배너 관리
1. "사이트 관리 > 배너" 클릭
2. "새 배너 등록" 버튼 클릭
3. 배너 이미지 업로드 또는 HTML 코드 입력
4. 배너 그룹 선택 (상단, 사이드, 하단 등)
5. 노출 기간 및 페이지 설정
6. 저장 후 배너 통계 확인

#### 팝업 관리
1. "사이트 관리 > 팝업" 클릭
2. "새 팝업 등록" 버튼 클릭
3. 팝업 제목 및 내용 작성 (HTML/CSS 편집 가능)
4. 노출 설정 (기간, 페이지, 빈도)
5. 팝업 옵션 설정 (오늘 하루 보지 않기 등)
6. 미리보기 후 저장

#### 다국어 관리
1. "사이트 관리 > 다국어" 클릭
2. 지원 언어 추가 (한국어, 영어, 일본어 등)
3. 시스템 메시지 다국어 등록
4. 콘텐츠별 다국어 버전 관리 (페이지, 메뉴 등)

#### 파일 관리
1. "사이트 관리 > 파일 관리" 클릭
2. 폴더 구조 탐색 및 파일 검색
3. 파일 업로드 (단일/다중, 드래그 앤 드롭)
4. 파일 수정, 삭제, 권한 설정
5. 스토리지 용량 확인 및 정리

### 6. 통계
```
GNB: 통계
LNB: 사용자 / 게시글
```

#### 사용자 통계
1. "통계 > 사용자" 클릭
2. 기간 선택 (일별, 주별, 월별)
3. 방문자 수, 페이지뷰, 유입 경로 확인
4. 디바이스 분석 및 회원 가입 추이 확인
5. 보고서 다운로드 (PDF, Excel)

#### 게시글 통계
1. "통계 > 게시글" 클릭
2. 게시판별 게시글 현황 확인
3. 인기 게시글 조회 (조회수, 댓글수 기준)
4. 게시글 작성 추이 및 댓글 통계 확인

### 7. 시스템 관리
```
GNB: 시스템 관리
LNB: 게시판 템플릿 / 게시판 동적 필드 관리
```

#### 게시판 템플릿 관리
1. "시스템 관리 > 게시판 템플릿" 클릭
2. "새 템플릿 생성" 버튼 클릭
3. 템플릿 이름 및 설명 입력
4. 기본 템플릿 선택 후 커스터마이징 또는 새로 작성
5. 스킨 파일 업로드 (목록/보기/쓰기/수정 페이지)
6. 미리보기로 확인 후 저장
7. 게시판 생성 시 해당 템플릿 선택 가능

#### 게시판 동적 필드 관리
1. "시스템 관리 > 게시판 동적 필드 관리" 클릭
2. "새 필드 생성" 버튼 클릭
3. 필드 정보 입력:
   - 필드명 (영문)
   - 필드 레이블 (한글)
   - 필드 타입 (텍스트, 숫자, 날짜, 선택 등)
4. 필드 검증 설정 (필수 여부, 길이 제한 등)
5. 필드 권한 설정 (읽기/쓰기 권한)
6. 저장 후 게시판 설정에서 필드 적용
