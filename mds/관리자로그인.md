# 관리자 로그인 구현

## 구현 완료 사항

### 1. 로그인 기능
- **로그인 페이지**: `/login`
- **로그아웃**: `/logout`
- **기본 리다이렉션**: `/` → `/admin/dashboard`

### 2. 보안 설정 (SecurityConfig)
- Spring Security를 이용한 폼 로그인
- 비밀번호 암호화 (BCryptPasswordEncoder)
- 역할 기반 접근 제어 (ROLE_ADMIN, ROLE_MANAGER)
- CSRF 보호 활성화

### 3. 사용자 인증 (CustomUserDetailsService)
- DB에서 사용자 정보 조회 (userId 기준)
- 사용자 활성화 상태 확인 (useYn = 'Y')
- 사용자 레벨에 따른 권한 부여:
  - ADMIN → ROLE_ADMIN
  - MANAGER → ROLE_MANAGER
  - 기타 → ROLE_USER

### 4. 자동 관리자 계정 생성 (AdminInitService)
애플리케이션 시작 시 기본 관리자 계정 자동 생성

**기본 관리자 정보:**
- 아이디: `admin`
- 비밀번호: `admin1234`
- 이름: 시스템 관리자
- 권한: ADMIN

### 5. 관리자 화면
#### 로그인 페이지
- 모던한 디자인 (그라데이션 배경)
- 에러 메시지 표시
- 로그아웃 성공 메시지
- 기본 계정 정보 안내

#### 대시보드 (/admin/dashboard)
- 사용자 정보 표시
- 통계 카드 (게시판, 메뉴, 사용자, 방문자)
- 빠른 메뉴 링크
- 최근 활동 표시

#### 레이아웃 (fragments/layout.html)
- 고정 사이드바 네비게이션
- 활성 메뉴 하이라이트
- 반응형 디자인
- 로그아웃 버튼

## 사용 방법

### 1. 데이터베이스 준비
MySQL 데이터베이스와 사용자 생성:
```sql
CREATE DATABASE cms CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER 'cms'@'localhost' IDENTIFIED BY 'cms1234';
GRANT ALL PRIVILEGES ON cms.* TO 'cms'@'localhost';
FLUSH PRIVILEGES;
```

### 2. 애플리케이션 실행
```bash
./gradlew bootRun
```

### 3. 로그인
1. 브라우저에서 `http://localhost:8080` 접속
2. 로그인 페이지로 자동 리다이렉션
3. 기본 관리자 계정으로 로그인:
   - 아이디: `admin`
   - 비밀번호: `admin1234`

### 4. 관리자 화면 사용
- 로그인 후 대시보드로 이동
- 왼쪽 사이드바에서 메뉴 선택
- 오른쪽 상단에서 로그아웃 가능

## 주요 엔드포인트

| URL | 설명 | 인증 필요 |
|-----|------|----------|
| `/login` | 로그인 페이지 | X |
| `/logout` | 로그아웃 | O |
| `/` | 홈 (대시보드로 리다이렉션) | O |
| `/admin/dashboard` | 관리자 대시보드 | O (ADMIN/MANAGER) |
| `/admin/menus` | 메뉴 관리 | O (ADMIN/MANAGER) |
| `/admin/boards` | 게시판 관리 | O (ADMIN/MANAGER) |
| `/admin/users` | 사용자 관리 | O (ADMIN/MANAGER) |

## 권한 체계

- **ROLE_ADMIN**: 전체 관리 권한
- **ROLE_MANAGER**: 제한된 관리 권한
- **ROLE_USER**: 일반 사용자 권한
